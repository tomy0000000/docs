# ✈️ Flying

Everything about flying.

## Platforms

- [Google Flight](https://www.google.com/flights/)
- [Skyscanner](https://www.skyscanner.com/)
- [KAYAK](https://www.kayak.com/)
- [Kiwi.com](https://www.kiwi.com/)
- [Going](https://www.going.com/): Flight deal alert based on crawled Google Flight data
- [Priceline](https://www.priceline.com/): Offer "Express Deal" which are slightly cheaper but don't know exact itinerary until booked.
- [Navan](https://navan.com/): Corporate travel platform but also do leisure
