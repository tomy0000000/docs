import { Icon } from '@iconify/react';
import { PackageManagerList } from '@site/src/components/PacageManagerCommands';

This page defines the fundation of the package manager ecosystem.

| Pacakage Manager                                                  | CLIs           | Installable                    | Source | Global | Project | Website                                                |
| ----------------------------------------------------------------- | -------------- | ------------------------------ | ------ | ------ | ------- | ------------------------------------------------------ |
| <Icon icon={PackageManagerList[0].icon} inline={true} /> Debian   | `apt`          | Package                        | PPA    | ✅     | ❌      | [Ubuntu Packages Search](https://packages.ubuntu.com/) |
| <Icon icon={PackageManagerList[1].icon} inline={true} /> Homebrew | `brew`         | Formula / Formulae / Cask [^1] | Tap    | ✅     | ❌      | [Homebrew](https://formulae.brew.sh/)                  |
| <Icon icon={PackageManagerList[2].icon} inline={true} /> Python   | `pip`          | Module                         | URL    | ✅     | ❌      | [PyPI](https://pypi.org/)                              |
| <Icon icon={PackageManagerList[3].icon} inline={true} /> npm      | `npm`          | Package                        |        | ✅     | ✅      | [npm](https://www.npmjs.com/)                          |
| <Icon icon={PackageManagerList[4].icon} inline={true} /> Ruby     | `gem`, `bndle` | Gem                            | URI    | ✅     | ✅      | [RubyGems](https://rubygems.org/)                      |
| <Icon icon={PackageManagerList[5].icon} inline={true} /> Go       | `go`           | Package                        |        |        |         | [Go Packages](https://pkg.go.dev/)                     |

[^1]: Homebrew refers to its installables as `formula`/`formulae`, which is normally a CLI tool, and `cask` for everything else, like apps, fonts, and such.
