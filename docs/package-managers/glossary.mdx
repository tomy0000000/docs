This page defines the fundation of the package manager ecosystem.

| Pacakage Manager                                                                | Installable                    | Source | Global | Project | Website                                                |
| ------------------------------------------------------------------------------- | ------------------------------ | ------ | ------ | ------- | ------------------------------------------------------ |
| <Icon icon={PackageManagerList[0].icon} inline={true} /> Debian (`apt`)         | Package                        | PPA    | ✅     | ❌      | [Ubuntu Packages Search](https://packages.ubuntu.com/) |
| <Icon icon={PackageManagerList[1].icon} inline={true} /> Homebrew               | Formula / Formulae / Cask [^1] | Tap    | ✅     | ❌      | [Homebrew](https://formulae.brew.sh/)                  |
| <Icon icon={PackageManagerList[2].icon} inline={true} /> Python (`pip`)         | Module                         | URL    | ✅     | ❌      | [PyPI](https://pypi.org/)                              |
| <Icon icon={PackageManagerList[3].icon} inline={true} /> npm                    | Package                        |        | ✅     | ✅      | [npm](https://www.npmjs.com/)                          |
| <Icon icon={PackageManagerList[4].icon} inline={true} /> Ruby (`gem`, `bundle`) | Gem                            | URI    | ✅     | ✅      | [RubyGems](https://rubygems.org/)                      |

[^1]: Homebrew refers to its installables as `formula`/`formulae`, which is normally a CLI tool, and `cask` for everything else that are not maintained by Homebrew core team.
