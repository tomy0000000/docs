# 國際匯款

| 匯款方式                   | 匯款行費用 | 中轉行費用 | 受款行費用 |
| -------------------------- | ---------- | ---------- | ---------- |
| BEN                        | 受款人     | 受款人     | 受款人     |
| SHA, 一般匯款, General     | 匯款人     | 受款人     | 受款人     |
| OUR, 全額到行, Pay in Full | 匯款人     | 匯款人     | 受款人     |
| 全額入戶, Receive in Full  | 匯款人     | 匯款人     | 匯款人     |
