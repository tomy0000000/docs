# Website Keyboard Shortcut

Useful keyboard shortcuts for popular websites.

## Google Calendar

- View Keyboard Shortcuts: <kbd>⌘</kbd> + <kbd>/</kbd>
- Previous period: <kbd>K</kbd> / <kbd>P</kbd>
- Next period: <kbd>J</kbd> / <kbd>N</kbd>

## Google Meet

- View Keyboard Shortcuts: <kbd>⌘</kbd> + <kbd>/</kbd>
- Toggle Mute: <kbd>⌘</kbd> + <kbd>D</kbd>
- Toggle Chat: <kbd>Ctrl</kbd> + <kbd>⌘</kbd> + <kbd>C</kbd>

## YouTube

- View Keyboard Shortcuts: <kbd>⌘</kbd> + <kbd>/</kbd>
- Previous/Next 10 seconds: <kbd>←</kbd> <kbd>→</kbd>
- Previous/Next Frame: <kbd>,</kbd> <kbd>.</kbd>
- Speed Down/Up: <kbd>&lt;</kbd> <kbd>&gt;</kbd>
- Toggle Caption: <kbd>C</kbd>
- Toggle Fullscreen: <kbd>F</kbd>
- Toggle Mute: <kbd>M</kbd>
- Toggle Miniplayer: <kbd>I</kbd>

## Google Maps

- View Keyboard Shortcuts: <kbd>⌘</kbd> + <kbd>/</kbd>
- Zoom In/Out: <kbd>+</kbd> <kbd>-</kbd>
- Move Around: <kbd>↑</kbd> <kbd>↓</kbd> <kbd>←</kbd> <kbd>→</kbd>

## LinkedIn

- View Keyboard Shortcuts: <kbd>?</kbd>
- Search: <kbd>/</kbd>
- Focus Previous/Next Post: <kbd>J</kbd> <kbd>K</kbd>
