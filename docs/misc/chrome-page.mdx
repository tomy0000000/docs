# Chrome Page

List of useful Chrome utility pages, mostly available in other Chromium-based browsers as well.

| URL                          | Description                                                                      |
| ---------------------------- | -------------------------------------------------------------------------------- |
| chrome://chrome-urls/        | List of all Chrome URLs                                                          |
| chrome://flags/              | Experimental features                                                            |
| chrome://net-internals/#hsts | HSTS settings, useful when you want the browser to forget a site's HSTS settings |
| chrome://net-internals/#dns  | DNS query. Like `dig` CLI, but within the browser.                               |
| chrome://settings/passkeys   | Managing stored passkeys. Some chromium-based browsers intentionally hide this.  |
| chrome://policy/             | Policy settings                                                                  |
| chrome://apps/               | Installed web apps                                                               |
