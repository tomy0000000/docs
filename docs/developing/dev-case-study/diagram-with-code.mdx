# Diagram

Draw diagrams with code!

## Language & Syntax

- [Mermaid](https://mermaid.js.org/): JavaScript-based, GitHub native support, and [Live editor](https://mermaid.live/) makes it easily accessible.
- [D2](https://github.com/terrastruct/d2): JavaScript-based, rising star.
- [Graphviz](https://graphviz.org/): Written in C, more powerful, but tricky to use.
- [PlantUML](https://plantuml.com/): Java-based, old-school, but powerful.

## Wrappers

- [Diagrams](https://github.com/mingrammer/diagrams): Python library with some predefined components, use Graphviz under the hood.
