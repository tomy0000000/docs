# ðŸ“Ÿ CLIs

A list of common CLI commands and how they're installed on my system.

<SortableTable
  columns={[
    {
      header: "CLI",
      accessorFn: (row) => [row.names, row.url],
      cell: ({ getValue }) => {
        const [names, url] = getValue();
        return url ? (
          <a href={url} target="blank" rel="noreferrer noopener">
            {names
              .map((name, index) => <code key={index}>{name}</code>)
              .reduce((prev, curr) => [prev, ", ", curr])}
          </a>
        ) : (
          names
            .map((name, index) => <code key={index}>{name}</code>)
            .reduce((prev, curr) => [prev, ", ", curr])
        );
      },
    },
    {
      header: "Category",
      accessorKey: "category",
    },
    {
      header: "Completion (zsh)",
      accessorKey: "zsh-completion",
      cell: ({ getValue }) => {
        const completion = getValue();
        return completion ? completion : "?";
      },
    },
    {
      header: "Alias",
      accessorKey: "alias",
      cell: ({ getValue }) => {
        const alias = getValue();
        return alias ? <code>{getValue()}</code> : null;
      },
    },
  ]}
  data={clis}
/>

## macOS Zsh Completion

- Built-in completions:
  - `/usr/share/zsh/5.9/functions`
  - `/usr/share/zsh/site-functions`
- Homebrew completions:
  - `/usr/local/share/zsh/site-functions`
  - `/usr/local/share/zsh-completions` (When `zsh-completions` formula is installed)
