## Exploring

### Dependency

List all dependencies of an installable.

In another word, find out a package depend on which other packages.

<PacageManagerCommands commands={commands.dependency} />

:::warning

For `npm`, you need to install the package first before you can list its dependencies.
In addition, [this command is expected change in the future](https://docs.npmjs.com/cli/v10/commands/npm-ls#note-design-changes-pending).

:::

### Information

Show or view the information of an installable.

<PacageManagerCommands commands={commands.information} />

### Reverse Dependency

List all dependent installables of a installable.

In another word, find out which installables are using / depend by a installable specified.

<PacageManagerCommands commands={commands.reverseDependency} />

### Search

Find an installable.

<PacageManagerCommands commands={commands.search} />

:::caution

[PyPI no longer supports 'pip search' (or XML-RPC search)](https://warehouse.pypa.io/api-reference/xml-rpc.html#deprecated-methods)

:::

## Source

### Add Source

Add sources to the package manager to install packages from the new source.

<PacageManagerCommands commands={commands.addSource} />

### List Sources

List all sources where installables can be installed from.

<PacageManagerCommands commands={commands.listSource} />

:::info

If the `pip` command yield `ERROR: No such key`, it means the source is the default index url, which is https://pypi.org/simple.

:::
