## Web Safe Fonts

Fonts that are pre-installed by many operating systems.

- `sans-serif`
  - Arial
  - Verdana
  - Tahoma
  - Trebuchet MS
- `serif`
  - Times New Roman
  - Georgia
  - Garamond
- `monospace`
  - Courier New
- `cursive`
  - Brush Script MT
