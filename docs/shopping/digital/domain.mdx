# 🌐 Domain

How to buy a domain name.

## 📋 Checklist

- [Internet Archive](https://archive.org/): Check previous site. Empty pages may be a suspicious sign.
- [Google Transparancy Report](https://transparencyreport.google.com/copyright/explore?copyright_data_exploration=q:;ce:domain): Search for take-down history.
- [The Lumen database](https://lumendatabase.org/): Search for copyright complaints.
- [Ahrefs](https://ahrefs.com/): Search for SEO history (may require paid account).
