## Components

### 床

- Bed Frame: 床架
  - Headboard: 床頭板，好看用的
  - Footboard: 床尾板，好看用的，不一定會有，有的話通常會和床頭板搭配
  - Bed Frame Cover: 床架套，好看用的
  - Box Spring / Foundation / Bed Base: 床板，通常是床墊下面的支撐板，可能內建於床架，也可能需要另外購買
  - Bed Legs / Casters: 床腳 / 輪子，床架下面的支撐腳，有的床架可能會有輪子
- Mattress: 床墊

### 棉被 / 被套 / 枕頭

:::info

棉被有兩種組成方式：

- Flat Sheet + Comforter: 美國常見的組合，直接用一張被單墊在身體和棉被之間
- Duvet + Duvet Cover: 其他地區常見的組合，用棉被套包住整個棉被

:::

- Bed Sheets
  - Fitted Sheet: 床單，用來包住床墊，避免直接接觸床墊
  - Flat Sheet: 被單，用來墊在棉被下面，避免直接接觸棉被
- Comforter (Duvet / Duvet Insert) / Quilt / Blanket: 棉被 / 比較厚的棉被 / 毛毯
  - Duvet Cover: 棉被套，用來包住棉被，避免直接接觸棉被
- Pillows: 枕心
  - Pillowcase: 枕頭套，用來包住枕心，避免直接接觸枕心
  - Pillow Sham: 枕頭套，好看用的，不適合用來睡覺，不一定會有，有的話通常會和床單搭配。
- Mattress Topper: 薄墊，相當於薄片版的床墊，通常如果床墊老化失去支撐力，可以墊在上面增加舒適度。
