## Dell

Dell monitor models are coded as follows:

```
{SERIES}{INCH}{YEAR}{RESOLUTION}{FEATURES}
```

- Series
  - `U`: UltraSharp, better color accuracy for professional use
  - `P`: P-Series, for productivity
  - `S`: S-Series, budget
  - `E`: Essential, entry-level
  - `AW`: Alienware, gaming-focused
  - `C`: Conference, typically comes with built-in webcam
- Resolution
  - `H`: FHD widescreen (1080p, 1920x1080)
  - `D`: WQHD (1440p, 2560x1440)
  - `W`: UWQHD (3440x1440)
  - `Q`: 4K UHD (2160p, 3840x2160)
  - `K`: 8K
- Features
  - `C`: with USB-C connectivity
  - `F`: AMD FreeSync
  - `G`: Nvidia G-Sync / Graphics
  - `T`: Touchscreen
