## E-bike Motors

|                | Hub Motor                   | Mid-Drive Motor            |
| -------------- | --------------------------- | -------------------------- |
| Motor Location | In the wheel hub            | Near the bike's crankset   |
| Cost           | Lower                       | Higher                     |
| Top Speed      | Generally lower             | Generally higher           |
| Maintenance    | Easier to maintain          | More complex maintenance   |
| Weight         | Heavier                     | Better weight distribution |
| Best For       | Casual riders, flat terrain | Hilly terrain, performance |
